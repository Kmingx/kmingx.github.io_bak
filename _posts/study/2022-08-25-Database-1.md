### Database
### Database
SQL 데이터 베이스에 저장된 데이터를 조회, 입력, 수정등의 조작이나 테이블 및 객체를 생성 및 제어하는 역할

SQL 의 종류

- DDL : 데이터베이스 구조를 정의한 언어
- DML : 데이터 조작을 위한 언어
- DCL  : 시스템 트랜잭션을 관리 하기 위한 언어
